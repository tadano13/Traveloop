# Traveloop

Traveloop is a modern travel planning application built with React.

## Features
- 🔐 Secure Login & Signup
- 🗺️ Trip Planning & Management
- 🎒 Packing Checklist
- 📓 Travel Journal
- 💰 Budget Tracker
- 🏙️ Multi-city Itineraries

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
