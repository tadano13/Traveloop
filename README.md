# 🌍 Traveloop - Professional Travel Planning App

Traveloop is a premium, feature-rich travel planning application designed to help travelers organize their journeys with ease and style. From budget tracking to itinerary building and memory journaling, Traveloop is the ultimate companion for your next adventure.

## ✨ Features

- **📊 Smart Dashboard**: Get a birds-eye view of your travel stats, upcoming trips, and quick tools.
- **🗺️ Itinerary Builder**: Detailed planning for travel, accommodation, and activities with real-time cost estimation.
- **💰 Budget Tracker**: Set trip budgets and track expenses by category (Transport, Hotel, Food, etc.) with visual progress bars.
- **📋 Packing Checklist**: Never forget an item again with categorized packing lists and progress tracking.
- **✍️ Travel Journal**: Record your memories and track your mood for every trip.
- **🔍 Destination Discovery**: Explore popular global destinations and add them directly to your trips.
- **🔐 Secure Authentication**: Full user account management powered by Supabase Auth.
- **🎨 Premium UI/UX**: Modern, responsive design using Tailwind CSS and Lucide React icons.

## 🚀 Tech Stack

- **Frontend**: React.js (v19)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Backend/Database**: Supabase (PostgreSQL)
- **Routing**: React Router (v7)

## 🛠️ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- A Supabase project

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/tadano13/Traveloop.git
   cd Traveloop
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Environment Setup**:
   Create a `.env` file in the root directory and add your Supabase credentials:
   ```env
   REACT_APP_SUPABASE_URL=your_supabase_url
   REACT_APP_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the application**:
   ```bash
   npm start
   ```

## 📂 Project Structure

- `src/pages`: Main application views (Dashboard, Itinerary, Budget, etc.)
- `src/components`: Reusable UI components like Navbar and Layout.
- `src/lib`: External library configurations (Supabase client).
- `src/index.css`: Global styles and Tailwinds configuration.
